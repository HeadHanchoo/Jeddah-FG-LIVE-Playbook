# Jeddah Flag Football Website

Static website for the Jeddah Flag Football 5v5 team.

## What is included

- Team landing page using the logo colors
- Roster section
- Searchable Jeddah playbook with cropped diagram images from the reference PDF
- Interactive play designer using a 70 x 30 yard field model
- Local browser saving for one custom play

## GitHub Pages Upload

1. Upload everything in this folder to a GitHub repository.
2. In GitHub, open Settings > Pages.
3. Set the source to the main branch and the root folder.
4. Open the published GitHub Pages URL after deployment finishes.

The site does not need a backend server.
